# Roadmap

## Near term

- Named saved combinations.
- Persistent roll history.
- Refined hand-drawn sprite atlas.
- Two-D10 visual presentation for D100.
- Dedicated Cardputer ADV motion calibration.
- More complete hardware test matrix.

## Later

- Advantage and disadvantage.
- Keep/drop highest and lowest dice.
- Exploding dice.
- Success-pool rules.
- Character and campaign profiles.
- Import/export through microSD.
- Optional wireless sharing and VTT integrations.

The core roller should remain fast and system-agnostic as these features grow.
