# Contributing / Contribuindo

Thank you for helping improve DICE\\VESSEL.

## Bug reports

Please include:

- Cardputer model and revision;
- firmware version;
- selected language and relevant settings;
- dice expression;
- exact reproduction steps;
- expected and observed behavior;
- photo or video when the problem involves motion, sound, or display timing.

Keep feature requests focused on the project goal: a fast, simple, system-agnostic campaign companion.

## Relatos de problemas

Inclua:

- modelo e revisão do Cardputer;
- versão do firmware;
- idioma e configurações relevantes;
- expressão de dados;
- passos exatos para reproduzir;
- comportamento esperado e observado;
- foto ou vídeo quando o problema envolver movimento, som ou tempo da animação.

Mantenha sugestões alinhadas ao objetivo do projeto: um companheiro de campanha rápido, simples e independente de sistema.

## Development

1. Create a focused branch.
2. Build with `pio run -e m5stack-cardputer`.
3. Test click-to-roll even when working on IMU behavior.
4. Verify both English and Portuguese UI text.
5. Keep unrelated changes out of the same pull request.
