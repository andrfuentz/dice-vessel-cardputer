# DICE\\VESSEL v0.4.0-beta

**KEEP ROLLING.**

The first public beta of DICE\\VESSEL turns the M5Stack Cardputer into a compact RPG campaign companion. Build flexible rolls with a guided workflow, save recurring combinations, review session history, and roll by key or motion.

This beta includes English and Brazilian Portuguese, black-and-amber visuals, animated dice physics, procedural wooden-box audio, a dedicated coin presentation, critical/failure feedback, and an animated charging screen.

## Recommended download

Use `dicevessel-v0.4-beta-factory.bin` for a complete flash at offset `0x0`.

## Test focus

- Cardputer original and ADV compatibility.
- Motion sensitivity and accidental shake detection.
- Audio volume across different hardware units.
- Guided input flow and saved combinations.
- Charging detection and battery percentage reporting.

Please report the Cardputer model, firmware build, expression used, and steps to reproduce any problem.
